### Tech Stack

- Minumum SDK 26
- Kotlin
- Coroutines
- Retrofit2 + Gson
- OkHttp3
- Room
- Kotlin Flow
- Dagger-Hilt
- SplashScreen
- WorkManager
- AlarmManager
- JUnit4
- Material Components
- Timber
