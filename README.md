# 반찬의민족 - 반찬 주문 앱
(전설의 모험가) Skipancho🌮가 만들고 (얼음 학살자) Iceman🧊이 배달하는 우아한 반찬 주문 앱 **반찬의민족**

### Tech Stack

- Minumum SDK 26
- Kotlin
- Coroutines
- Retrofit2 + Gson
- OkHttp3
- Room
- Kotlin Flow
- Dagger-Hilt
- SplashScreen
- WorkManager
- AlarmManager
- JUnit4
- Material Components
- Timber
